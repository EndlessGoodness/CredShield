CredShield - Financial Advisor Website
======================================
VB .NET Windows Forms Application

PROJECT STRUCTURE:
==================

✓ LoanManager.vb - Global Module
  └─ Manages SelectedLoanType variable across forms

✓ Form1.vb - WELCOME PAGE
  ├─ Welcome title with company tagline
  ├─ Description about CredShield
  ├─ "Get Started" button → navigates to Form2
  └─ "Exit" button → closes application

✓ Form2.vb - LOADING PAGE
  ├─ Animated loading screen with progress bar
  ├─ Shows loading percentage (0-100%)
  ├─ Auto-transitions to Form3 after completion
  └─ Professional blue/gray design

✓ Form3.vb - LOGIN PAGE
  ├─ Professional header with company branding
  ├─ Email Address input field
  ├─ Password input field (masked)
  ├─ "Remember Me" checkbox
  ├─ Login validation (non-empty fields)
  ├─ "Login" button → navigates to Form4 (Home)
  ├─ "Sign Up" link
  └─ "Back" button → returns to Form1

✓ Form4.vb - HOME PAGE (MAIN DASHBOARD) - ENHANCED WITH DETAILED DESCRIPTIONS
  ├─ Header Section:

  ├─ Loan Type Selection Panel (3 Buttons):
  │  ├─ "HOME LOANS" button (Blue) → Form5 with Home Loan details

  │  └─ "FINANCIAL LOANS" button (Purple) → Form5 with Financial Loan details
  │
  └─ Main Content Section:
     ├─ COMPREHENSIVE About Section:
     │  └─ "A reliable and user-friendly platform designed to simplify the process 
     │     of accessing loans and financial services..."
     │  └─ Includes mission statement and company values
     │
     ├─ Detailed Services Section (Full-Width Cards):
     │
     ├─ 🏠 HOME LOAN – Build Your Dream Home
     │  └─ Complete description of home loan services
     │
     ├─ 🏦 BANK LOAN – Trusted and Secure Lending
     │  └─ Complete description of bank loan services
     │
     ├─ 💰 FINANCIAL LOAN – Manage Your Personal Needs
     │  └─ Complete description of financial loan services
     │
     └─ Why Choose CredShield? (Features list)

✓ Form5.vb - LOAN REGISTRATION PAGE
  ├─ Dynamic Form based on Loan Type:
  │
  ├─ For HOME LOANS:

  ├─ For BANK LOANS:
  │  ├─ Loan Amount
  │  ├─ Loan Tenure
  │  ├─ Annual Income
  │  ├─ Employment Status
  │  ├─ Purpose of Loan
  │  └─ Repayment Mode
  │
  ├─ For FINANCIAL LOANS:
  │  ├─ Loan Amount
  │  ├─ Loan Tenure
  │  ├─ Annual Income
  │  ├─ Employment Status
  │  ├─ Business/Investment Type
  │  └─ Project Details
  │
  ├─ Common Fields (all forms):
  │  ├─ Contact Number
  │  ├─ Email Address
  │  └─ Terms & Conditions checkbox
  │
  ├─ Form Validation:
  │  ├─ All required fields must be filled
  │  ├─ Numeric validation for amounts and income
  │  └─ Terms must be accepted

  ├─ "Submit Application" button → navigates to Form6

✓ Form6.vb - RESULT/CONFIRMATION PAGE

  ├─ Application Details Box:
  │  ├─ Generated Reference Number (Format: CSH-YYYYMMDD-XXXXXX)
  │  └─ Loan Type Display
  ├─ Next Steps Information:
  │  ├─ Document verification process
  │  ├─ Email/SMS update notifications
  │  ├─ Approval timeline (3-5 business days)
  │  └─ Disbursement process guidance
  ├─ "Go to Home" button → returns to Form4
  └─ "New Application" button → returns to Form4

APPLICATION FLOW:
=================

Start
  ↓
Form1 (Welcome) → "Get Started" ↓
Form2 (Loading) → Auto-transition ↓
Form3 (Login) → "Login" ↓
Form4 (Home Dashboard) → Select Loan Type ↓
  ├─ "HOME LOANS" → Form5 (Home Loan Registration)
  ├─ "BANK LOANS" → Form5 (Bank Loan Registration)
  └─ "FINANCIAL LOANS" → Form5 (Financial Loan Registration)
  ↓
Form5 (Registration) → Fill Form & "Submit" ↓
Form6 (Confirmation) → "Go to Home" or "New Application" ↓
Form4 (Home Dashboard) → Can select another loan type or logout

ENHANCED SERVICE DESCRIPTIONS:

🏠 HOME LOAN – Build Your Dream Home
   At CREDSHIELD, our Home Loan services are designed to help you turn your dream 
   of owning a home into reality. Whether you are purchasing a new house, constructing 
   a property, or renovating your existing home, we provide flexible loan options with 
   easy repayment plans.
   
   We focus on making the process simple and transparent, ensuring quick approvals and 
   minimal documentation. With competitive interest rates and customer-friendly terms, 
   CREDSHIELD supports you at every step of your homeownership journey.

🏦 BANK LOAN – Trusted and Secure Lending
   Our Bank Loan services at CREDSHIELD are designed to provide users with reliable 
   and secure loan options in collaboration with trusted banking systems. These loans 
   are ideal for larger financial needs such as business expansion, investments, or 
   long-term planning.
   
   We ensure that all bank loan processes are handled with high security, proper 
   verification, and transparency. CREDSHIELD acts as a bridge between users and 
   structured banking services, making loan access easier and more efficient.

💰 FINANCIAL LOAN – Manage Your Personal Needs
   CREDSHIELD offers Financial Loans to help you meet your personal and urgent 
   financial requirements. Whether it's for education, medical expenses, travel, or 
   any other personal need, our financial loans provide quick access to funds when 
   you need them the most.
   
   Our system ensures a smooth and hassle-free application process, allowing users 
   to apply easily and receive fast responses. With flexible repayment options and 
   secure processing, we aim to make financial support accessible to everyone.

TESTING INSTRUCTIONS:
=====================
1. Run the application - starts at Form1 (Welcome Page)
2. Click "Get Started" - goes to Form2 (Loading)
3. Wait for loading to complete (or watch progress bar)
4. Fill login form with any email and password
5. Click "Login" - goes to Form4 (Home Dashboard)
6. Review the comprehensive service descriptions on home page
7. Click one of three loan type buttons:
   - HOME LOANS (Blue) - Shows home-specific fields
   - BANK LOANS (Green) - Shows bank-specific fields
   - FINANCIAL LOANS (Purple) - Shows business-specific fields
8. Fill the registration form completely
9. Check the terms checkbox
10. Click "Submit Application" - goes to Form6 (Confirmation)
11. View the generated reference number and application details
12. Click "Go to Home" to return to Form4 and apply for another loan
13. Click "Logout" in Form4 header to return to Form1
